<h1 align="center">How to run the application:</h1>
<h4>Requirements</h4>
<ul>
  <li>Node: v14.16.0
  <li>NPM: v6.14.11
  <li>MongoDB: v4.4
</ul>

<h4>Steps</h4>
<ul>
<li>at the terminal, clone the project:</li>

```git clone https://github.com/carlosvght/navedexAPI.git```

<li>enter in the folder project:</li>

```cd navedexAPI```

<li>install the dependencies:</li>

```npm install```

<li>run server:</li>

```npm run dev```
</ul>
